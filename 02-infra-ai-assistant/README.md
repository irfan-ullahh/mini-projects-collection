# INFRA – Intelligent Nexus Framework for Responsive Assistance

A text-based AI Assistant built in pure Java as my 2nd semester OOP project at UET Peshawar.

## Features
- Contextual keyword parsing
- Dynamic response mapping using HashMap
- Randomized replies for natural conversation
- Pure OOP implementation (Encapsulation, Inheritance, Polymorphism)

## How to Run
```bash
javac Main.java
java Main
